# ConvertTo-Shared

## SYNOPSIS
Converts a Cloud User Mailbox to a Shared Mailbox, Disables the AD User & Removes any licenses

## SYNTAX

```
ConvertTo-Shared [[-UserToConvert] <String>]
```
